<h4 align="left">💳  Projeto Conta Bancária - Java Sistema simples de gerenciamento de contas bancárias, desenvolvido com foco em estudos de Programação Orientada a Objetos (POO) utilizando Java.</h4>

###

<h4 align="left">1.📌 O Projeto Conta Bancária é um sistema de gestão projetado para simular e administrar operações financeiras relacionadas a contas bancárias. Oferece funcionalidades como cadastro, consulta, atualização e remoção de contas, além de transações como depósitos, saques e transferências.</h4>

###

<p align="left">Classes e Objetos;<br>Atributos e Métodos;<br>Encapsulamento;<br>Herança;<br>Polimorfismo;<br>Abstração.</p>

###

<h4 align="left">2.Funcionalidades</h4>

###

<p align="left">Criar Conta (Corrente ou Poupança)<br>Listar todas as Contas<br>Consultar Conta por número<br>Consultar Conta por titular<br>Atualizar dados da Conta<br>Excluir Conta<br>Saque<br>Depósito<br>Transferência entre contas</p>

<h4 align="left">6. Como Executar o projeto no Eclipse/STS</h4>

###

<p align="left"></p>

###

<h6 align="left">6.1. Importando o Projeto</h6>

###

<p align="left">1.Clone o repositório do Projeto [Conta Bancária](https://github.com/BrunoAlves-tech/conta-bancaria-springboot)<br>dentro da pasta do Workspace do Eclipse/STS</p>

###

<p align="left">git clone https://github.com/BrunoAlves-tech/conta-bancaria-springboot.git</p>

###

<p align="left">2.Abra o Eclipse/STS e selecione a pasta do Workspace onde você clonou o repositório do projeto<br>3.No menu superior do Eclipse/STS, clique na opção: File 🡲 Import...<br>4.Na janela Import, selecione a opção: General 🡲 Existing Projects into Workspace e clique no botão Next<br>5.Na janela Import Projects, no item Select root directory, clique no botão Browse... e selecione a pasta do Workspace onde você clonou o repositório do projeto<br>6.O Eclipse/STS reconhecerá automaticamente o projeto<br>7.Marque o Projeto Conta Bancária no item Projects e clique no botão Finish para concluir a importação</p>

###

<h6 align="left">6.2. Executando o projeto</h6>

###

<p align="left">1.Na guia Package Explorer, localize o Projeto Conta Bancária<br>2.Abra a Classe Menu<br>3.Clique no botão Run source: imgur.com para executar a aplicação<br>4.Caso seja perguntado qual é o tipo do projeto, selecione a opção Java Application<br>5.O console exibirá o menu do Projeto</p>
